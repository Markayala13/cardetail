**Steps:**

2. Click "Add Property" and enter: `https://figueroasmobiledetail.com`

- Request indexing for the homepage
- Monitor coverage and performance weekly

4. Upload photos from assets/img/ folder
5. Post updates regularly
6. Respond to all reviews

### 3. Bing Webmaster Tools

**Steps:**
1. Go to [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. Add site: `https://figueroasmobiledetail.com`
3. Import data from Google Search Console (if available)
4. Submit sitemap: `https://figueroasmobiledetail.com/sitemap.xml`

### 4. Google Analytics 4 (GA4)

**Setup:**
1. Create account at [Google Analytics](https://analytics.google.com)
2. Create property for website
3. Get Measurement ID (format: G-XXXXXXXXXX)
4. Add tracking code to index.html before `</head>`:

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```
